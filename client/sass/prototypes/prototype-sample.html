Sample prototype

You only have a few requirements to setting up your sample prototype:

  • Name it in the prototypes array in your huron.config.js.
  • Create a prototypes folder inside one of your KSS folders (that also should listed in your Huron config).
  • Name the prototype file should be prototype-[name].html.
  • Make sure to wrap the contents of the prototype with a single root element (e.g. div).
